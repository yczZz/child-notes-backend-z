# 宝宝记录后端 - Wiki 首页

## 项目简介

**child-notes-backend** 是一款面向微信小程序的宝宝日常记录后端服务，提供喂养、睡眠、换尿布、体温、成长等多种记录类型的管理与统计功能。

## 功能概览

| 模块 | 说明 |
|------|------|
| 认证 | 微信小程序登录、JWT Token 鉴权、用户资料管理 |
| 宝宝管理 | 多宝宝档案 CRUD，自动计算月龄 |
| 记录管理 | 12 种记录类型（喂养/睡眠/尿布/体温/补充剂/成长/异常/吸奶/辅食/疫苗/活动/里程碑） |
| 统计 | 今日概览、按日/按范围统计、发烧腹泻检测与消退标记 |
| 文件上传 | 阿里云 OSS 存储，支持图片等文件上传 |
